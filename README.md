# Premium Side-Bar | STUDIOx

A sleek, modern, and highly responsive sidebar navigation menu built with HTML, CSS, and Vanilla JavaScript. 

## 🌟 Features

- **True Glassmorphism:** Features a premium frosted glass effect (`backdrop-filter`) that beautifully overlays background imagery.
- **Dark Luxury Aesthetic:** Carefully chosen color palette, typography (Outfit font), and subtle gradients to ensure a high-end look and maximum legibility.
- **Fluid Micro-Animations:** Includes smooth hover states, dynamic active indicators, and scalable interactive icons.
- **Accessible & Robust:** Fully keyboard accessible (Escape to close), semantic HTML5 tags, and overlay-click to close functionality.
- **Zero Dependencies:** No heavy frameworks—just pure, optimized HTML, CSS, and JS.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure (`<aside>`, `aria-labels`).
- **CSS3:** Custom properties (variables), Flexbox, Transitions, Glassmorphism.
- **JavaScript (ES6):** Clean, lightweight logic for DOM manipulation and event handling.
- **Font Awesome:** For scalable vector icons.
- **Google Fonts:** Utilizing the 'Outfit' font family.

## 🚀 Getting Started

Simply clone this repository and open `index.html` in your favorite web browser. No build steps or package managers required!

```bash
git clone https://github.com/yourusername/mini-sidebar.git
cd mini-sidebar
```
